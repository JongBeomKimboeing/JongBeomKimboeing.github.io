# 1. 크롤링이란?
**웹페이지에서 필요한 데이터를 추출해내는 작업**
- 크롤링하는 프로그램: 크롤러
- 크롤링은 웹페이지의 정보를 HTML문서로 표현한다.

<br>
<br>

# 2. 크롤링을 위해 필요한 것
웹페이지의 HTML을 얻기 위해 request 라이브러리를 이용하고<br>
가져온 HTML을 분석하기 위해 BeutifulSoup 라이브러리를 사용한다.


<br>
<br>
